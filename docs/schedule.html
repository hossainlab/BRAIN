BRAIN Training Program Timeline

Overview

The BRAIN course is structured as a comprehensive training program with 16 modules covering fundamental to advanced topics in neurogenomics, single-cell analysis, and AI applications.

Module Organization

Phase 1: Foundations (Weeks 1-3)

Focus: Building foundational knowledge in neurogenomics, single-cell technology, and essential computational skills.

  • Introduction to neurogenomics and its importance in brain research
  • Understanding single-cell RNA sequencing technology
  • Mastering Linux command-line tools
  • Learning the CellRanger pipeline
  • Developing Python programming skills

Phase 2: Core Analysis Methods (Weeks 4-6)

Focus: Learning essential single-cell data analysis techniques.

  • Quality control and filtering of single-cell data
  • Dimensionality reduction and visualization
  • Data integration across datasets
  • Clustering and cell population identification
  • Differential expression analysis
  • Cell type annotation and prediction

Phase 3: Advanced Topics (Weeks 7-8)

Focus: Exploring advanced analysis methods and AI applications.

  • Trajectory inference and pseudotime analysis
  • Spatial transcriptomics analysis
  • Deep learning methods for single-cell genomics
  • Neural networks, autoencoders, and transformers
  • Integration of AI in neuroscience research

Session Format

Each module typically includes:

  1. Lecture Component (30-40%)
    • Theoretical foundations
    • Method explanations
    • Real-world applications
  2. Hands-on Tutorial (50-60%)
    • Guided coding exercises
    • Working with real datasets
    • Step-by-step implementation
  3. Discussion & Q&A (10-20%)
    • Questions and clarifications
    • Best practices
    • Troubleshooting
